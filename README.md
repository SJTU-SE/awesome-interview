## 项目介绍

受[awesome-se](https://github.com/SJTU-SE/awesome-se)项目的启发，我们创立了一个收集上海交通大学软件学院同学在实习或者求职面试时的面经，收集的目的在于为后来的同学提供一些参考，也希望你能够积极地提交自己的面试经历到这个项目中。如果你有贡献的想法可以直接 Pull Requests。
